# 🦁 LionPOS

Sistema Punto de Venta moderno y escalable construido con tecnologías de vanguardia.

## ✨ Características principales
- 🛒 Gestión completa de ventas
- 📦 Control de inventario en tiempo real
- 👥 Administración de clientes (CRM)
- 📊 Reportes y dashboard interactivo
- 🔐 Autenticación multi-rol (admin/cajero)
- ☁️ Arquitectura 100% cloud native

## 🛠️ Stack tecnológico
**Frontend:** React 18 + Vite + TailwindCSS  
**Backend:** ASP.NET Core 10 Web API  
**Base de datos:** Azure SQL Database  
**Imágenes:** Cloudinary CDN
